LibreTV搭建在线影视网站

2025-05-28 2 min read # 影音 # 在线 # LibreTV # GitHub # 开源

项目简介

Github高热度开源项目LibreTV，是一个轻量级、免费的在线视频搜索与观看平台，提供来自多个视频源的内容搜索与播放服务。无需注册，即开即用，可以直接部署到Netlify、Cloudflare、Vercel或者Docker等平台。

项目主页：https://github.com/LibreSpark/LibreTV

快速部署

首先准备Cloudflare账号、Github账号，可自行到官网注册。

  • 1、Fork 或克隆LibreTV仓库到自己的 GitHub 账户

https://github.com/LibreSpark/LibreTV

  • 2、登录Cloudflare后台，进入 Pages 服务

选择 计算(Workers) -> Workers和Pages -> 创建Pages -> 导入现有 Git 存储库。

  • 3、将Github账户和Cloudflare进行关联，然后会显示刚Fork的项目。

  • 4、其他设置不动，建议环境变量添加PASSWORD并设置密码，以控制访问权限规避版权

  • 5、保存并部署，之后Cloudflare会自动分配一个域名，浏览器直接访问即可。

使用效果

打开网站后建议点击右上角的设置按钮，把所有源都勾选上，黄色过滤按钮打开。

其他事项

  • 此教程仅演示Cloudflare的部署方法，你也可以部署到Netlify、Vercel等其他平台，包括本地NAS也可以通过Docker部署，具体步骤可参考官方教程。
  • 播放器支持以下键盘快捷键：

空格键: 播放/暂停
左右箭头: 快退/快进
上下箭头: 音量增加/减小
M 键: 静音/取消静音
F 键: 全屏/退出全屏
Esc 键: 退出全屏

  • 所有视频均来自第三方 API 接口提供的搜索结果，如有侵权内容，请联系相应的内容提供方。
  • 同类型的开源影视聚合平台还有MoonTV，部署方式大同小异，具体请查看官方教程。
